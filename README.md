## Управление
    - W/S – вперёд/назад
    - A/D – поворот корпуса
    - Q/E – наклон стрелы
    - Z/X – наклон клешни
    - Space – открыть/закрыть клешню
	- Tab - включить/выключить автопилот
	- F - перейти на FPV-камеру
